# 🚀 PaieCashFan v7.0 - Super App Mondiale

## 📊 Vue d'ensemble
Super application multinationale pour les fans de sport avec système de récompenses blockchain, wallet crypto, et réseau social intégré.

## ✨ Fonctionnalités Actuelles
- ✅ Interface type Instagram/TikTok avec Stories et Feed
- ✅ Système FOMO avec cashback PaieCash Coins (PCC)
- ✅ Envoi d'argent sécurisé avec code PIN
- ✅ Stories dynamiques par club
- ✅ Feed social avec interactions (like, comment, share)
- ✅ Navigation bottom bar
- ✅ Wallet integration

## 🌐 URLs
- **Local Dev**: http://localhost:3000
- **Production**: À déployer sur Cloudflare Pages

## 🔧 Stack Technique
- **Framework**: Hono v4.11
- **Runtime**: Cloudflare Workers
- **Frontend**: Vanilla JS + TailwindCSS
- **Database**: Cloudflare D1 (à configurer)
- **Storage**: Cloudflare KV/R2 (à configurer)

## 📦 Installation
```bash
npm install
```

## 🚀 Développement
```bash
# Build le projet
npm run build

# Démarrer avec PM2
pm2 start ecosystem.config.cjs

# Tester
curl http://localhost:3000
```
