================================================================
                    FONCTIONS DISPONIBLES
================================================================

1. genererCarteHTML(joueur, type)
   - Genere HTML d'une carte (FAN ou VIP)
   - Parametres : objet joueur, type ('fan' ou 'vip')
   - Retour : HTML string

2. genererPageJoueur(joueurKey)
   - Genere page complete pour un joueur
   - Parametres : cle joueur (ex: 'hakimi')
   - Retour : HTML complet

3. chercherPhotoJoueur(nomJoueur, clubJoueur)
   - Cherche photo joueur sur Transfermarkt
   - Parametres : nom, club
   - Retour : URL photo

4. genererCartesClub(nomClub)
   - Genere toutes les cartes d'un club
   - Parametres : nom club
   - Retour : array objets cartes

================================================================
                    UTILISATION
================================================================

POUR AJOUTER UN NOUVEAU JOUEUR :

1. Trouver photo sur Transfermarkt
2. Trouver logo club sur Wikipedia
3. Ajouter dans joueursDatabase :

'nouveaujoueur': {
    nom: 'NOM COMPLET',
    club: 'Nom Club',
    pays: 'Pays',
    drapeau: 'emoji',
    poste: 'Poste',
    photoUrl: 'URL_PHOTO',
    logoClub: 'URL_LOGO',
    numeroCarteFan: '5412 7534 9876 XXXX',
    numeroCarteVip: '5412 7534 9876 XXXX'
}

4. Generer page : genererPageJoueur('nouveaujoueur')
